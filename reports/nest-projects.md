# NestJS Projects

Para criar um novo projeto `NestJS` execute o seguinte comando:

Obs.: Projetos em `NestJS` precisam usar `TypeScript` 

```bash
nest new <project_name>
```
Estrutura básica de um projeto `NestJS`:

```bash
<project-name>/
├── src/
│   ├── app.controller.ts
│   ├── app.module.ts
│   ├── app.service.ts
│   └── main.ts
├── test/
├── node_modules/
├── package.json
├── tsconfig.json
└── .eslintrc.js
```
